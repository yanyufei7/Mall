# fruitmall

> An application to buy fruit

## 构建安装程序

``` bash
# 安装依赖
npm install

# 启动项目，浏览器访问 http://localhost:8080/
npm run dev
```

## 技术栈
- Vue
- VueRouter
- Vuex
- Axios
- Stylus
- Vant

## 模块
`- [x]` 首页
`- [x]` 账号登陆
